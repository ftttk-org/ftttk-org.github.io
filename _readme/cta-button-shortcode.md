# CTAボタンショートコード (CTA Button Shortcode)

## 📋 概要

どのページでも使用できるセンタリングされたCTA（Call to Action）ボタンのショートコードです。統一されたデザインで、レスポンシブ対応の美しいボタンを簡単に追加できます。

## 🎯 実装日

**2025年10月13日**

## 🔧 実装内容

### ファイル
- **ショートコード**: `layouts/shortcodes/cta-button.html`
- **記録**: `_records/cta-button-shortcode.md`

### 機能
- センタリングされたCTAボタン
- 3つのスタイルバリエーション
- 3つのサイズバリエーション
- レスポンシブ対応
- ホバー効果とアニメーション
- アクセシビリティ対応

## 📝 使用方法

### 基本的な使い方
```markdown
{{< cta-button >}}
```

### パラメータ付きの使い方
```markdown
{{< cta-button 
  text="ボタンテキスト" 
  url="/リンク先/" 
  style="primary" 
  size="large" 
  target="_self" 
>}}
```

## 🎨 パラメータ一覧

| パラメータ | デフォルト値 | 説明 | オプション |
|-----------|-------------|------|-----------|
| `text` | "お申し込みはこちら" | ボタンのテキスト | 任意の文字列 |
| `url` | "/admission/" | リンク先URL | 任意のURL |
| `style` | "primary" | ボタンスタイル | `primary`, `secondary`, `outline` |
| `size` | "large" | ボタンサイズ | `small`, `medium`, `large` |
| `target` | "_self" | リンクの開き方 | `_self`, `_blank` |
| `class` | "" | 追加CSSクラス | 任意のクラス名 |

## 🎨 スタイルバリエーション

### Primary（プライマリ）
- **背景**: 紫系グラデーション
- **色**: 白文字
- **用途**: メインのCTA、重要なアクション

### Secondary（セカンダリ）
- **背景**: 白
- **ボーダー**: 紫
- **色**: 紫文字（ホバー時は白）
- **用途**: サブのCTA、補助的なアクション

### Outline（アウトライン）
- **背景**: 透明
- **ボーダー**: 紫
- **色**: 紫文字（ホバー時は白）
- **用途**: 控えめなCTA、情報提供

## 📏 サイズバリエーション

### Small（スモール）
- **パディング**: 0.5rem 1.5rem
- **フォントサイズ**: 0.9rem
- **用途**: コンパクトな表示

### Medium（ミディアム）
- **パディング**: 0.75rem 2rem
- **フォントサイズ**: 1rem
- **用途**: 標準的な表示

### Large（ラージ）
- **パディング**: 1rem 2.5rem
- **フォントサイズ**: 1.1rem
- **用途**: 目立たせたいCTA
